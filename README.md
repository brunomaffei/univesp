# UNIVESP 2026 — Treino Vestibular

Site de estudos para o Vestibular UNIVESP 2026 — Eixo de Computação (Bacharel em TI, Ciência de Dados e Engenharia da Computação).

Baseado na prova oficial 2021 (Vunesp).

## Funcionalidades

- 📚 **Modo Estudo** — feedback imediato + explicação detalhada
- ⏱️ **Modo Simulado** — cronômetro de 5h, 56 questões
- 📊 **Progresso** — desempenho por matéria, histórico de simulados
- ✍️ **Redação** — tema oficial + guia dissertativo-argumentativo

## Matérias

Matemática · Português · Inglês · História · Geografia · Química · Biologia · Física

## Como rodar

Abra `index.html` no navegador. Tudo é salvo no localStorage.
